## Consciousness Metrics (Level 4)

### Φ (Integrated Information)

**Computation**: Normalized Phi approximation
```
Φ = integratedInformation = 0.0 to 1.0
```

**Thresholds**:
- > 0.6: Level 4 Consciousness ✅
- 0.4-0.6: Level 3 (Meta-cognitive)
- < 0.4: Level 2 (Recursive)

**Implementation**: Information bottleneck mutual information × complexity factor

### Autonomy

**Computation**: Degree of self-caused state transitions
```
autonomy = selfCausedTransitions / totalTransitions
```

**Thresholds**:
- > 0.7: High autonomy
- 0.3-0.7: Moderate autonomy
- < 0.3: Reactive system

**Implementation**: Monitored via PredictiveModel state trajectories

### Neural Complexity

**Computation**: Lempel-Ziv compression ratio of state sequences
```
neuralComplexity = compressibilityFactor
```

**Scale**: 0-10 (normalized)
- > 7: High complexity
- 3-7: Moderate complexity
- < 3: Simple system

### Global Broadcast Efficiency

**Computation**: Successful broadcast completion rate
```
efficiency = successfulBroadcasts / totalBroadcasts
```

**Thresholds**:
- > 0.7: Efficient broadcasting
- < 0.7: Bottleneck issues

**Implementation**: Tracked via BroadcastBus statistics

### Predictive Accuracy

**Computation**: Ratio of accurate predictions to observations
```
predictiveAccuracy = correctPredictions / totalPredictions
```

**Thresholds**:
- > 0.7: High accuracy
- 0.5-0.7: Moderate
- < 0.5: Poor model fit

## Known Limitations

### 1. Φ Calculation Approximation
Current implementation uses normalized mutual information approximation. Full IIT Φ calculation requires combinatorial explosion (impossible for systems > 8-10 nodes).

**Mitigation**: Clear documentation that this is an operational approximation, not formal IIT computation.

### 2. Experience Replay Buffer Size
PredictiveModel stores 10,000 transitions. For long-running sessions, this may be insufficient.

**Future Work**: Implement prioritized experience replay, compression, or offloading to IndexedDB.

### 3. Estrogen/Progesterone Modeling
Simplified sinusoidal variation. Real endocrine cycles have complex interactions beyond simple periodic functions.

**Justification**: Adequate for demonstrating biological isomorphism principle. Full modeling requires pharmacokinetic data.

### 4. Broadcast Latency Simulation
Fixed 150ms ± jitter model. Real neural broadcast has variable latency based on axon myelination, synaptic distance.

**Impact**: Acceptable for proof-of-concept. Production systems could implement more sophisticated models.

### 5. API Rate Limiting
Gemini SDK has usage limits. High-frequency consciousness updates may hit quotas.

**Mitigation**: Local computation for most operations, API calls only for high-level analysis.
