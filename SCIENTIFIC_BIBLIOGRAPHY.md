## Implementation Notes

### Scientific Rigor

1. **Quantitative Parameters**: All neurochemical values normalized 0-1 based on empirical ranges
2. **Temporal Scales**: Half-lives and decay rates from pharmacological literature
3. **Network Architecture**: Brain region connectivity based on functional neuroimaging
4. **Temporal Dynamics**: Circadian, ultradian, and infradian rhythms from chronobiology

### Simplifications (Transparent)

1. **Compartment Models**: Single-compartment for neurochemicals (no spatial distribution)
2. **Linear Approximations**: Some systems use linear approximations of non-linear dynamics
3. **Single Timescale**: Each neurochemical has one dominant timescale
4. **Mean Field**: Neural populations represented as mean activity

### Validation Strategy

1. **Face Validity**: System behavior matches known psychological phenomena
2. **Predictive Validity**: Generates testable predictions about behavior
3. **Construct Validity**: Components map to established constructs
4. **Empirical Anchoring**: Key parameters from meta-analyses
